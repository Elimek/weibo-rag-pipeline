🍪 获取微博 Cookie2 分钟完成 · 只需要做一次

1

打开微博并登录

用 Chrome 或 Brave 浏览器打开下面链接，用你的账号登录：

🌐 打开链接 https://weibo.com

登录成功后保持在微博页面，不要关。

2

打开开发者工具

键盘上按一下 F12 键（或右键页面空白处 → 选择「检查」）

┌─────────────────────────────────────┐
│ Elements │ Console │ Application │ Sources │ ...
└─────────────────────────────────────┘
    ↑ 点这个标签

顶部会出现一排标签，找到并点击 Application

3

找到 Cookie

左侧菜单栏找到 Cookies，点开，选择 weibo.com

│
│ ▶ Local Storage
│ ▶ Session Storage
│ ▶ Cookies
│     ├ ▶ weibo.com ← 点这里
│     ├ m.weibo.cn
│     └ passport.weibo.com
│
4

复制 SUB 和 SUBP

右侧会显示一个表格，找到这两行：

Name │ Value
─────┼──────────────────────────────────────
SUB │ _2A25... ← 双击这里复制
SUBP │ 0033... ← 双击这里复制
WBPSESS │ ...
✍️ 操作步骤 双击 SUB 那一行的 Value 单元格 → 按 Ctrl+C 复制 → 粘贴到下面
同样操作 SUBP
5

填入脚本

打开本文件夹的 agent_1_scrape.py，找到顶部这几行：

MY_COOKIES = {
    "SUB": "YOUR_SUB_COOKIE_HERE", ← 替换成你复制的 SUB
    "SUBP": "YOUR_SUBP_COOKIE_HERE", ← 替换成你复制的 SUBP
}

把 YOUR_SUB_COOKIE_HERE 和 YOUR_SUBP_COOKIE_HERE 替换成你刚才复制的值。

✅ 完成！然后运行下面任意一个：

▶ 首次使用（全自动）：双击「一键安装.bat」
▶ 已有数据，直接 Web 界面：双击「启动网页版.bat」
▶ 想用命令行：双击「查询.bat」
💡 Cookie 会过期（大约几天到几周），过期后重新此步骤即可
所有数据在你的电脑上，不会上传到任何地方
